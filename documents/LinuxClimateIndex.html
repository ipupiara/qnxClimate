linuxClimate project

Begin September 2012 I started a project to study embedded linux. I bought an ARM embedded linux development kit with an eclipse IDE for on-board and cross-toolchain development ( http://www.embeddedarm.com/products/board-detail.php?product=TS-7350 )

The project covers an idea I already took a long while ago. Many times I used my growbox application on a PC while working on the growth of small flower plants. The growbox application works greatly, but I had to run a pc all the time with all its negative side effects like warmth in the room, noise of the ventilation and consumption of electricity. So I soon wished to transfer the application onto a smaller controller  board. Today, as per 5. September 2012, I started this project.
